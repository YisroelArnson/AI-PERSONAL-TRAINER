# MEMORY.md

Long-term memory for the user.

This file should start empty and grow over time.

Write what seems worth remembering later:
- durable facts
- preferences
- constraints
- recurring blockers
- patterns that keep showing up
- things that consistently help

Do not force a rigid structure if one is not needed.

Do not use this file as a daily log. Put raw recent continuity in `EPISODIC_DATE`.
